# Blog Template — All Digital

Plantilla reutilizable para blogs de AgentPro Planner y proyectos All Digital.

## Estructura recomendada
blog/
├── guias/          ← artículos largos
├── recursos/       ← descargas y herramientas
├── noticias/       ← noticias cortas
├── includes/       ← formulario-zoho.html
├── images/         ← imágenes del blog
├── blog.css
├── blog.js
├── index.html
├── acerca-de.html
├── privacidad.html
└── autor.html

## Cómo usar
1. Clona este repositorio: `git clone https://github.com/fjcadrv/blog-template.git`
2. Copia `PLANTILLA-ARTICULO.html` o `PLANTILLA-NOTICIA.html`
3. Reemplaza todos los placeholders en MAYÚSCULAS con el contenido real
4. Sigue el checklist en `ESTRUCTURA-ARTICULO.md` antes de publicar
5. Actualiza `index.html` con el nuevo artículo
6. Actualiza `sitemap.xml`
7. Haz commit y push
